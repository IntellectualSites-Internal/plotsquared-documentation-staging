1. Introduction

Permission packs are pre-selected, bundled permissions helping server managers to assign a package rather 30+ unique nodes.

  • Basic permission pack: plots.permpack.basic

  • Basic Flag Pack: plots.permpack.basicflags

  • Basic Inbox Pack: plots.permpack.basicinbox
